The **Quantum Retro Composer** is a Python-based audio engine that generates unique, copyright-free, seamless background music for retro-style video games (SNES, Genesis, Indie).

Unlike standard random generators, this tool uses a sophisticated pipeline to ensure musicality and "soul":
1.  **Quantum Entropy:** It harvests real-time quantum vacuum fluctuations from the ANU Quantum Numbers API to seed the generator.
2.  **AI Composition:** It uses **Google's Gemini 2.5 Flash** to write the "sheet music" (JSON), handling music theory, syncopation, and song structure.
3.  **DSP Synthesis:** It renders the audio from scratch using pure mathematics (NumPy), applying retro oscillators (Square/Triangle/Saw waves) and bit-crushing effects.
